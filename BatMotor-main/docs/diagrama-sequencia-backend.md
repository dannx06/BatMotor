# Diagrama de sequência — API BatMotor (backend)

## draw.io / diagrams.net

Abre o ficheiro **`diagrama-sequencia-backend.drawio`** (na mesma pasta `docs/`) em:

- [diagrams.net](https://app.diagrams.net/) → **Abrir** → escolher o ficheiro, ou  
- VS Code / Cursor com extensão **Draw.io Integration**.

O ficheiro tem **duas páginas** (separadores em baixo no editor):

1. **Login POST /auth/login** — fluxo completo com nota sobre alternativas 400/401.  
2. **POST /movimentacao** — fluxo opcional (middleware, controller, service, Prisma).

---

## Código XML

O XML completo está em **`docs/diagrama-sequencia-backend.drawio`**. Podes copiar o conteúdo para um novo `.drawio` se precisares.
